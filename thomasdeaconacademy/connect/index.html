This connection configuration was created by TDA. For more information, see Thomas Deacon Academy.

Connect to the Thomas Deacon Academy RDS network with Microsoft Remote Desktop using the connection configuration file above to access a Windows desktop environment on your device. The configuration allows the connection to the TDA RDS network through connect.thomasdeaconacademy.peterborough.sch.uk which is the external access point for the remote desktop protocol. The connection to TDA is over HTTPS which ensures security in the transmission of your data between the server and your client.

This requires Microsoft Remote Desktop, which is installed by default in Windows and can be downloaded from the Mac App Store on OS X. An iPad app and an Android app are currently in development.
