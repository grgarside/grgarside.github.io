Wide Stack Overflow & Stack Exchange

Stack Exchange 2015-03-31

About the Wide Stack Overflow and Stack Exchange Stylesheet

With the recent design changes across Stack Exchange, making CSS stylesheets that work across Stack Exchange is very easy. I’ve finally decided that now’s a good time to redevelop a ‘wide’ CSS stylesheet to make a wide Stack Overflow and therefore a wide Stack Exchange. This stylesheet extends the width of Stack Overflow, Super User and all other Stack Exchange sites which have been updated to the latest CSS. This also works on all meta sites such as Meta Stack Exchange and Meta Stack Overflow.

Wide Stack Overflow extends the width of the page to 1330px with padding, leaving 1300px for the content. This fits the content with sufficient padding for my MacBook Pro with Retina display at Best for Retina scaling (equivalent to 1440 × 1900). Feel free to change the CSS to make this even wider for 1920 × 1080 displays. As well as wide Stack Overflow content, this stylesheet also widens other areas of Stack Exchange, including the revision history, 10k tools, review queues, and more!
