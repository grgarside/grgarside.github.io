Frequently Asked Casio fx Calculator Questions

What is Casio's natural display on fx-991ES and fx-82ES?

The natural textbook display is a way of displaying formulae in the closest possible way to handwriting. For example, fractions are displayed vertically. This makes it easy to write and edit formulae naturally without the complicated brackets. It is enabled by default on supporting calculators, including fx-82ES and fx-991ES. You can enable \(\text{MathO}\) mode using \(\color{#A80}{\fbox{SHIFT}\ \fbox{SETUP}}, \fbox{1}\).

How to reset Casio fx-991ES or fx-82ES?

  1. Turn the calculator on with the \(\fbox{ON}\) button. There is no hardware reset button on the back.
  2. \(\color{#A80}{\fbox{SHIFT}}\ \fbox{9}\) to enter Reset. A list of items which can be reset are shown.
  3. \(\fbox{3}\) to select the \(\text{Reset All}\) option.
  4. \(\fbox{=}\) to confirm the reset. The calculator has been reset at this point.
  5. \(\fbox{AC}\) to return to normal calculation, or \(\color{#A80}{\fbox{SHIFT}}\ \fbox{AC}\) to turn the calculator off.

Is fx-991ES a programmable calculator?

No, the Casio fx-991ES PLUS is not a programmable calculator. The calculator can solve equations, but it is not programmable. This means the calculator is suitable for many exams, such as A-levels.
