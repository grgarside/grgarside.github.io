  • In any George Garside apps, iTunes Analytics is in effect. See iTunes Analytics privacy policy for more information.
  • Furthermore, in apps which contain iAd, data is collected per the iAd privacy policy.

Personally-identifiable information collected

  • Should the user authenticate with /thomasdeaconacademy, the user's username is recorded.
    This data is paired with the browser/OS pair, and AB testing if applicable, but no other information is correlated (e.g. IP address).
    Collection is carried out by KISSmetrics, using WhichBrowser's JS for browser/OS identification. See KISSmetrics privacy policy for more information. This information is not distributed with any third parties.
  • In Free Bitcoin, personally-identifiable information is stored per the user providing the information in the signup form. This information is not distributed with any third parties.
